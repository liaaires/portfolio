# Portfolio Lia Aires

Um portfólio desenvolvido com React, Vite e Tailwind CSS para exibir projetos de UI/UX, artigos e trajetória profissional.

## Como rodar o projeto localmente

1. Certifique-se de ter o **Node.js** instalado na sua máquina.
2. Instale as dependências executando o comando:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse o projeto no navegador através do endereço exibido no terminal (geralmente `http://localhost:5173`).

## Como publicar (Deploy)

Este projeto está configurado com **GitHub Actions**. Assim que você enviar os arquivos para a branch `main` ou `master` no GitHub, o site será automaticamente compilado e publicado no **GitHub Pages**.

> **Nota:** lembre-se de ir nas configurações do seu repositório no GitHub:
> 1. Vá em **Settings** > **Pages**
> 2. Em **Source**, mude para **GitHub Actions**.
